# TPI-PROG1-26
Trabajo colaborativo para resolucion de tpi-programacion1
El tpi lo dividimos en dos partes.
Desarrollo del sistema: Leonardo Astargo estuvo a cargo de las funciones 3, 5 y 6 del menú principal, mientras que Gonzalo Díaz desarrolló las opciones 1, 2 y 4. En cuato el menu y la aplicaión de colores fueron debatidas entre los dos participantes.

Instrucciones de uso: El programa es altamente intuitivo. Cada opción del menú guía al usuario detalladamente según el requerimiento. La navegación se realiza mediante las flechas del teclado e ingresando con la tecla Enter. Además, el sistema incluye funciones para pausar la pantalla ("presione una tecla para continuar") y limpiar la consola, evitando la acumulación de texto y optimizando la interfaz visual.
